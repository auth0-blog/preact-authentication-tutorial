## Installation

**1. Clone this repo:**

```sh
git clone https://github.com/auth0-blog/preact-authentication-tutorial
cd preact-authentication-tutorial
```

**2. Install the dependencies:**

```sh
npm install
```

**3. Run your app**

```sh
npm start
```
